# UUII Web - Universitas Utama Internasional Indonesia

Website kampus untuk Universitas Utama Internasional Indonesia yang berlokasi di Jalan Kaliurang No. 1144 KM 20, Yogyakarta.

## Program Studi
- **Teknik Industri**: Program studi yang fokus pada optimasi sistem terintegrasi
- **Pertanian**: Program studi yang fokus pada ilmu pengetahuan dan teknologi pertanian modern

## Teknologi
- HTML5
- CSS3 (dengan Flexbox dan Grid)
- JavaScript (Vanilla JS)

## Cara Menggunakan

### Langsung di Browser
Buka file `index.html` di browser web Anda.

### Menggunakan HTTP Server Lokal
```bash
# Menggunakan Python 3
python3 -m http.server 8080

# Atau menggunakan Node.js
npx http-server -p 8080
```

Kemudian buka `http://localhost:8080` di browser.
